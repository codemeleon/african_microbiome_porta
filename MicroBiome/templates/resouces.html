Microbiome analysis tools

Resouces Description Links
Tools for Microbiome analysis Useful R packages for 16S rRNA gene and shotgun data analysis and visualization https://microsud.github.io/Tools-Microbiome-Analysis/
Quantitative Insights Into Microbial Ecology (QIIME) QIIME is a bioinformatics pipeline enabling analysis of microbiome DNA metadata. It allows to perform statistical analyses of results and generate high-quality graphs https://qiime.org/
dada2 R package dada2 is an R package dedicated to the analysis of microbiome data http://benjjneb.github.io/dada2/

Microbiome related databases

Resouces Description Links
Human OncoBiome Database Database of comprehensive curated information on the bacterial species present in the human cancer microbiota http://www.juit.ac.in/hcmd/home
MicrobiomeDB MicrobiomeDB is a data mining platform enabling the exploration of microbiome studies. This platform also provides visualization and analysis tools to investigate microbiome data. https://microbiomedb.org/mbio/app/
MDB: Microbiome Database The microbiome Database gathers microbiome metadata. about the environment (oceanic, soil…) and host-associated microbiome. It also makes available the gene catalogs for host-associated microbiomes. https://db.cngb.org/microbiome/
Human Microbiome Project Data Portal The Human Microbiome Project Data Portal enables the exploration of studies and data associated with the human microbiome. Furthermore, this web interface provides access to the tools used to analyze microbiome. https://portal.hmpdacc.org/
Human Oral Microbiome Database (eHOMD) comprehensive and curated information o­n the bacterial species present in the human aerodigestive tract http://www.homd.org/
Disbiome database Database providing an overview of differential microbial composition in diseases https://disbiome.ugent.be/home
MDB: Microbiome Database This database provides sequencing resource and accurate metadata of metagenomics samples, as well as gene catalogs. https://db.cngb.org/microbiome/
Human Pan-Microbe Communities (HPMC) database Database of human gastrointestinal microbiota derived from faecal samples. http://www.hpmcd.org/index.php
